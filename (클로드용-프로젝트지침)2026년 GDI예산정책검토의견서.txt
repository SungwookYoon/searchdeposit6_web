# 국가예산안 검토 의견서 작성 프롬프트 지침

## 1. 문서 형식 및 구조

### 기본 문서 설정
```
- 문서 제목: "202X년 국가예산안 검토 의견서"
- 부제: "기관명/작성자명"
- 페이지 번호: 하단 중앙 "- X -" 형식
- 여백: 상하좌우 균등, A4 기준
```

### 목차 구성 (필수 4개 섹션)
```
1. 개인 고유 분야별 연구과제 제안 (23년 예산안과 연계)
2. 지역 일자리 증진 사업 제안 (고유 분야 및 타 분야)
3. 지역 전략사업 제안 (지역에서 전략적으로 추진할 사업)
4. 202X년도 사업 제안
```

## 2. 문체 및 표현 지침

### 문체 규칙
- **공문서체 사용**: 정중하고 격식있는 문체
- **경어법**: "~습니다", "~됩니다" 보다는 "~임", "~함" 사용
- **객관적 서술**: 1인칭 표현 지양, 3인칭 관점에서 서술
- **간결성**: 불필요한 수식어나 중복 표현 최소화

### 용어 사용 기준
```
- 정부 부처명: 정확한 공식 명칭 사용 (예: 산업통상자원부, 환경부)
- 예산 단위: "억원", "조원" 단위로 표기
- 법령 인용: 「법률명」 형식으로 표기
- 사업명: 정확한 공식 사업명 사용
```

## 3. 표 양식 및 레이아웃

### 박스형 요약 정보 (각 섹션 도입부)
```
[사업 제안 방향] 또는 [사업화 방향]
❍ 핵심 방향 1
❍ 핵심 방향 2  
❍ 핵심 방향 3
❍ 핵심 방향 4
```

### 개별 사업 제안 형식
```
□ 사업명 (예산 규모)
❍ 배경 및 주요 내용
­ (배경) 현황 및 필요성 설명
­ (주요내용) 구체적 추진 내용
❍ 관련 예산 사업
­ 부처명 사업명(예산규모), 연계 방안 설명
```

### 비교표 양식 (필요시)
```
| 구분 | 항목1 | 항목2 | 비고 |
|------|-------|-------|------|
| 내용 | 설명  | 설명  | 참고 |
```

## 4. 내용 구성 상세 지침

### 각 섹션별 구성 요소

**섹션 1: 개인 고유 분야별 연구과제**
- 분야 전문성 기반 제안 (2-3개 과제)
- 기존 국가예산과의 연계성 명확히 제시
- 지역 특화 요소 강조

**섹션 2: 지역 일자리 증진 사업**
- 직접 일자리 → 민간 일자리 창출 방향성
- 신산업 중심 인력 양성 방안
- 구체적 목표 수치 제시 (예: 3만명 플랫폼)

**섹션 3: 지역 전략사업**
- 국가 정책과의 정합성 강조
- 지역 고유 자원/인프라 활용 방안
- 중장기 비전과 연계

**섹션 4: 차년도 사업 제안**
- 연속성 있는 정책 추진 방안
- 중장기 계획 수립 포함

### 필수 포함 요소 (각 제안별)
```
1. 배경 및 현황 분석
2. 구체적 추진 내용
3. 예상 효과 및 성과 지표
4. 관련 국가예산 사업 매칭
5. 예산 규모 및 재원 조달 방안
```

## 5. 분량 및 페이지 구성

### 전체 분량 기준
- **총 6페이지 내외** (표지 제외)
- 섹션별 1-2페이지 배분
- 핵심 내용 위주로 간결하게 구성

### 페이지별 배분 가이드
```
페이지 1: 섹션 1 (개인 고유 분야) - 2-3개 과제
페이지 2-3: 섹션 2 (일자리 증진) - 3-4개 사업
페이지 3-4: 섹션 3 (지역 전략) - 4-5개 사업  
페이지 5-6: 섹션 4 (차년도 사업) - 2-3개 사업
```
